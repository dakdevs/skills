# dakdevs/skills

Agent skills for Claude Code.

## Install

```
npx skills add dakdevs/skills
```

## Skills

### tldraw-canvas

Interact with open [tldraw](https://tldraw.com) desktop canvases via a local HTTP API. Create, read, update, and delete shapes. Build diagrams, flowcharts, wireframes, and visual layouts programmatically.

### pr-this

Create or update a draft pull request for the current branch using `gh` CLI. Idempotent - always re-assesses the full diff and either creates a new PR or updates the existing one. Creates draft PRs by default.
